# flutter_timezone_plus

A fork of [flutter_timezone](https://github.com/tjarvstrand/flutter_timezone) works on windows and linux

## Usage examples

### Get the timezone
```dart
final String currentTimeZone = await FlutterTimezone.getLocalTimezone();
```
